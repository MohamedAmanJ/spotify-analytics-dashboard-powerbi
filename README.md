# 🎵 Spotify Analytics Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Spotify Analytics Dashboard developed using Power BI. The dashboard analyzes streaming performance, artist popularity, audio features, and listener trends to generate meaningful business insights.

## 📌 Key Insights

- 877 tracks generated approximately **464 Billion streams**.
- The Weeknd is the highest streamed artist.
- Average streams per track exceed **529 Million**.
- Major mode songs slightly dominate the dataset.
- Danceability and Energy show a positive relationship.
- Listener engagement has increased significantly after 2015.
